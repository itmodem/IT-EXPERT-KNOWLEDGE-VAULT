# Create Dropdown Menu in Excel

**Date:** 17-Jun-2026

**Category:** Excel & Spreadsheets

**Difficulty:** Beginner

## Problem

Needed a dropdown list for CRM status tracking.

## Solution

1. Select the cells.
2. Go to the **Data** tab.
3. Click **Data Validation**.
4. Select **List**.
5. Enter the values.
6. Click **OK**.

## Key Learning

Data Validation can create dropdown menus and standardize data entry.
