## What I Learned

Today I learned how to change a Windows 11 login password and how to create a strong password for better security.

A password is the first layer of protection for a computer. If a password is weak or predictable, unauthorized users may gain access to files, accounts, and sensitive business data.

## How to Change a Password in Windows 11

### Method 1: Using Settings

1. Press **Windows + I**.
2. Open **Accounts**.
3. Click **Sign-in Options**.
4. Select **Password**.
5. Click **Change**.
6. Enter the required information.
7. Save the changes.

## How to Create a Strong Password

A strong password should include:

- 16–20+ characters
- Uppercase letters (A–Z)
- Lowercase letters (a–z)
- Numbers (0–9)
- Special symbols (!, @, #, $, %, etc.)
- Random and unrelated words

### Avoid Using

- Personal names
- Company names
- Mobile numbers
- Birth dates
- Common words
- Simple patterns

### Strong Password Formula

Random Word + Symbol + Random Word + Number + Random Word + Symbol + Random Word

**Example Structure:**

`Word#Word84Word@Word`

## Key Learning

A strong password should be difficult for others to guess but easy for the owner to remember. Long passphrase-style passwords are generally more secure than short passwords.
